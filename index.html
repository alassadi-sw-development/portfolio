Portfolio Project 3 - Elvrum Online Shop

Elvrum Online Shop Website Project Features:

  1. Navigation Bar with Product Search: A dynamic navigation bar with a search function to find products easily, along with an interactive cart displaying the quantity of items.
  2. Order Summary Review: Review orders with the ability to update quantities and remove products from the cart.
  3. Delivery Date Calculation: Utilizes Date object to calculate delivery dates based on the order date, with varying delivery prices for each date.
  4. Payment Summary: Calculates the total sum of products, delivery costs, and taxes (MWST 19%).
  5. Rendering Functions: Includes rendering functions for products, order summary, and payment summary.
  6. Text Animation: Added text animation for the "Add to Cart" button.
